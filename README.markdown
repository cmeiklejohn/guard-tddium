Guard-TDDium
===============

Run your specs with tddium through guard whenever files are committed.

Usage 
=====

Install:

```ruby
gem guard-tddium
```

Setup:

```bash
guard init tddium
```

Requirements
============

One of the following three:

```ruby
  gem 'rb-inotify'
  gem 'rb-fsevent'
  gem 'rb-fchange'
```
